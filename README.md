# Task Tracker CLI (Java)

A simple Command Line Interface (CLI) application built with Java to manage tasks.
The application allows users to add, update, delete, and list tasks.
Tasks are persisted locally using a JSON file.

## Features
- Add new tasks
- Update task status (TODO, IN_PROGRESS, DONE)
- Delete tasks
- List all tasks
- Persistent storage using JSON file
- Clean architecture (Controller, Service, Repository)

## Technologies
- Java (JDK 17+)
- File I/O
- Enum for task status
- No external libraries

## Project Structure
```
src/
├── model/
├── repository/
├── service/
└── Main.java
```

## How to Run
1. Clone the repository
```bash
git clone https://github.com/your-username/task-tracker-cli.git
```
2. Open the project in IntelliJ IDEA
3. Run Main.java
